## Technologies

* **[Kotlin](https://kotlinlang.org/)** - primary programming language,
* **[Jetpack Compose](https://developer.android.com/jetpack/compose)** - modern declarative UI framework,
* **[Room](https://developer.android.com/training/data-storage/room)** - local SQLite database,
* **[Coroutines & Flow](https://kotlinlang.org/docs/coroutines-overview.html)** - asynchronous operations and reactive data streams,
* **[ViewModel & Navigation](https://developer.android.com/guide/navigation)** - UI state management and app routing.

---

## Features & Controls

* `Routes` - list of available cycling and running routes,
* `Saving` - long press the stop button to record and save your measured time to the database,
* `Timer` - tap the timer button to quickly pause or resume the current activity,
* `Tablet view` - seamless transition between smartphone (single-screen) and tablet (master-detail) layouts,
* `Themes` - switch between Light and Dark modes.
